# Chapter 01 - 데이터프레임 생성 & 조건 검색

### 1. 데이터 프레임 생성하기

외부 데이터셋인 `csv` 파일을 판다스 모듈을 이용해 데이터프레임 객체로 로드

```python
import pandas as pd

# emp.csv 파일을 읽어와 emp 데이터프레임 생성
emp = pd.read_csv('emp.csv')

# 한글 데이터가 섞인 파일을 불러올 때는 encoding 옵션 필요
# euckr : 한글을 이해하는 인코딩 옵션
emp = pd.read_csv('emp.csv', encoding='euckr')
```

> 💡 **인코딩(encoding)이란?** 기계가 이해하는 언어로 변환하는 것. 한글이 포함된 파일은 `euckr` 옵션을 추가해야 깨지지 않습니다.

---

### 2. 판다스의 핵심 구조

```python
emp          # 판다스 데이터프레임(DataFrame) → 오라클의 테이블
emp['sal']   # 판다스 시리즈(Series)          → 오라클의 컬럼
```

| 판다스    | 오라클 | 설명                          |
| --------- | ------ | ----------------------------- |
| DataFrame | 테이블 | 행과 열로 구성된 2차원 데이터 |
| Series    | 컬럼   | 하나의 열(1차원 데이터)       |

---

### 3. 특정 컬럼(열) 출력하기

원하는 컬럼만 선택해서 보고 싶을 때는 대괄호를 두 번 `[[ ]]` 감싸서 컬럼명을 지정

- **문법:** `데이터프레임명[['컬럼명1', '컬럼명2']]`

```python
# 사원 테이블에서 이름과 급여 컬럼만 선택 출력
emp[['ename', 'sal']]

# 모든 컬럼 출력
emp
```

---

### 4. 특정 조건에 해당하는 데이터 검색하기

조건에 맞는 행만 필터링할 때는 컬럼 선택과 검색 조건을 함께 사용

- **문법:** `데이터프레임명[['컬럼명1', '컬럼명2']][검색조건]`

```python
# 급여가 2000 초과인 사원의 이름과 급여 출력
emp[['ename', 'sal']][emp['sal'] > 2000]
```

#### 📌 기본 비교 연산자

| 연산자 | 의미   |
| ------ | ------ |
| `>`    | 초과   |
| `<`    | 미만   |
| `>=`   | 이상   |
| `<=`   | 이하   |
| `==`   | 같다   |
| `!=`   | 다르다 |

---

### 5. 자주 사용하는 기타 비교 연산자

SQL과 판다스의 비교 연산자 대응표

| 기능      | SQL                     | 판다스                                   |
| --------- | ----------------------- | ---------------------------------------- |
| 범위 검색 | `BETWEEN 1000 AND 3000` | `emp['sal'].between(1000, 3000)`         |
| 목록 포함 | `IN (10, 20)`           | `emp['job'].isin(['MANAGER', 'CLERK'])`  |
| NULL 검사 | `IS NULL`               | `emp['comm'].isnull()`                   |
| 패턴 검색 | `LIKE '%A%'`            | `emp['ename'].apply(lambda x: 'A' in x)` |

> ⚠️ **판다스에서 NOT은 `~`를 사용**
>
> ```python
> # NULL이 아닌 데이터 검색
> emp[~emp['comm'].isnull()]
>
> # 부서번호가 10, 20이 아닌 데이터 검색
> emp[~emp['deptno'].isin([10, 20])]
> ```

#### 💻 활용 예시

```python
# 1. 급여가 1000 이상 3000 이하인 사원
emp[emp['sal'].between(1000, 3000)]

# 2. 부서번호가 10 또는 20인 사원
emp[emp['deptno'].isin([10, 20])]

# 3. 커미션이 NULL인 사원
emp[emp['comm'].isnull()]

# 4. 이름에 'S'가 포함된 사원
emp[emp['ename'].apply(lambda x: 'S' in x)]
```

### 6. apply 함수와 lambda

컬럼(시리즈)의 데이터를 조건에 맞게 검색할 수 있도록 도와주는 함수

- **apply** : 각 행의 데이터에 조건을 하나씩 적용해서 검색
- **lambda** : 이름이 없는 한 줄짜리 함수를 만들 때 사용하는 키워드

#### 문법

```python
데이터프레임[['컬럼명']][데이터프레임['컬럼명'].apply(lambda x : 조건)]
```

#### lambda 동작 원리

```
emp['ename'].apply(lambda x : x[0] == 'A')
```

`emp['ename']`의 이름이 위에서부터 하나씩 `x`에 담기며 조건을 검사합니다.

```
x = 'KING'   →  x[0] = 'K'  →  'K' == 'A'  →  False
x = 'BLAKE'  →  x[0] = 'B'  →  'B' == 'A'  →  False
x = 'ADAMS'  →  x[0] = 'A'  →  'A' == 'A'  →  True ✅
```

> 💡 **파이썬 인덱스는 0부터 시작합니다**
>
> ```
> 'KING'[0]  →  K
> 'KING'[1]  →  I
> 'KING'[2]  →  N
> 'KING'[3]  →  G
> ```

#### 💻 활용 예시

```python
# 이름의 첫 번째 철자가 'A'인 사원의 이름 출력
emp[['ename']][emp['ename'].apply(lambda x: x[0] == 'A')]

# 이름의 세 번째 철자가 'A'인 사원
emp[['ename']][emp['ename'].apply(lambda x: x[2] == 'A')]

# 이름이 4글자인 사원
emp[['ename']][emp['ename'].apply(lambda x: len(x) == 4)]
```
